Atari 7800 Programming Tricks

In order to actually learn programming tricks for the Atari 7800, you will first need to learn the most basic things about the Atari 7800.




Hardware Specs

  • CPU: 6502C @ 1.79 MHz

    • Changes over the original 6502:
    • HALT signal on pin 35
    • R/W signal on pin 36
    • Stock 6502 has these, but aren't connected to any pins
  • GPU: Custom MARIA Graphics Chip

    • Hardware Information:
    • Total Resolution: 454x263@60Hz (NTSC), 454x313@50Hz (PAL)
    • Theoretical Visible Resolution: 320x243 (NTSC), 320x293 (PAL)
    • Practical Visible Resolution: 320x192 to 224 (NTSC), 320x240 to 288 (PAL)
    • Clock Speed: 7.16 MHz
    • Memory: Shared with the CPU
    • Video Modes:
    • Read Mode 0:
    • Write Mode 0 - 160-pixel, 2bpp (160A)
    • Write Mode 1 - 160-pixel, 4bpp (160B)
    • Read Mode 1: N/A
    • Read Mode 2:
    • Write Mode 0 - 320-pixel, faux 2bpp (320D)
    • Write Mode 1 - 320-pixel, 2bpp (320B)
    • Read Mode 3:
    • Write Mode 0 - 320-pixel, 1bpp (320A)
    • Write Mode 1 - 320-pixel, faux 2bpp (320C)
    • DMA Modes:
    • DMA Modes 0 and 1 - DO NOT USE, THESE DMA MODES CAN BREAK YOUR CONSOLE!!!
    • DMA Mode 2 - Normal DMA
    • DMA Mode 3 - No DMA
    • Halts CPU whenever it draws graphics due to DMA, except for DMA Mode 3.
