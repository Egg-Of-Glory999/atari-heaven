Atari 7800 Programming Tricks

In order to actually learn programming tricks for the Atari 7800, you will first need to learn the most basic things about the Atari 7800.




Hardware Specs

  • CPU: 6502C @ 1.79 MHz

    • Changes over the original 6502: HALT signal on pin 35, R/W signal on pin 36 (original 6502 has these, but aren't connected)
